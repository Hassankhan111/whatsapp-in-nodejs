# 📰 News Management System (Node.js)

A simple News Management System built with **Node.js**, **Express.js**, **EJS**, **MySQL**, and **Bootstrap**. This project allows administrators to manage news articles and users to view the latest news.

---

## 📌 Features

- User Registration & Login
- Secure Password Encryption
- Add News Articles
- Edit News Articles
- Delete News Articles
- View All News
- News Details Page
- Image Upload
- Responsive Design
- MySQL Database
- Session Authentication

---

## 🛠 Technologies Used

- Node.js
- Express.js
- MySQL
- EJS
- Bootstrap 5
- CSS3
- JavaScript
- Multer (Image Upload)
- bcryptjs (Password Hashing)
- Express Session
